Github+Hexo搭建个人博客

发表于 2016-09-18 | 分类于 BlogBuild | | 阅读次数:
字数统计: 1,073 | 阅读时长 ≈ 4

本文详细地说明了如何运用开源框架Hexo和使用Github免费的云空间完成个人免费博客的搭建，除此之外，还包括许多Hexo主题进行博客个性化设计的步骤．

一、知识了解

1．什么是Hexo

Hexo中文网站是一个快速，简洁且高效的博客框架．让上百个页面在几秒内瞬间完成渲染．Hexo支持Github　Flavored　Markdown的所有功能．甚至可以整合Octopress的大多数插件．并且自己也拥有强大的插件系统．简单理解Hexo就是一个开源的博客设计框架，个人可以随意进行下载、修改和设计．

２．什么是GIt和Github

通俗地解释，git是一种版本控制系统．跟svn、cvs是同级的概念．
github是一个网站，给用户提供git服务．这样你就不用自己部署git系统，直接用注册个账号，用他们提供的git服务就可以．可以理解为Git是在本地电脑上处理各种文件，而Github就像百度云盘云盘一样．更详细的解释可以参见大白话解释 Git 和 GitHub．

3．其他

诸如Node.js等都可以理解为一种程序语言解释器，不需要刻意掌握．

二、配置准备

1．Git

在Git官网下载属于本电脑配置的Git安装包并安装Git，安装过程比较简单，如有问题请参见Git安装配置教程．

2．Github

在Github官网注册Github账号，然后新建一个项目（PS：项目名称必须为:Github账号名.github.io，这也是设计的博客网址）．

3．Node.js

在Node.js官网下载后缀为LTS的安装包并安装．

三、本地搭建过程

1．安装Hexo.打开CMD命令行窗口(Ctrl+R)或者Windows PowerShell.运行如下命令行:

npm install -g hexo-cli

2．新建MyBlog文件夹,文件地址如:E:\MyBlog.并在该文件夹内右键点击Git Bash here.

3．依次输入并运行如下命令行:

hexo init
hexo install
hexo g
hexo s

这时便生成你的本地博客网页：https://localhost:4000，你便可以在浏览器中打开并预览你的博客网页（PS：如果你的电脑安装有福昕阅读器，会占用你的https://localhost:4000地址，此时你可以通过运行
hexo s -p 4001生成本地地址https://localhost:4001来预览你的本地博客。另外，请不要Crtl+C导致`hexo s`停止，否则无法打开你的本地博客网页。）

以上便是如何生成本地博客网页，建议你以后书写博文或者配置博客都先通过hexo g和hexo s两个命令在本地博客网页进行预览(一般修改配置后建议可先执行hexo clean清除缓存)，再部署到你的github上（也就是生成网页域名来查看你的博客网站，接下来便是如何进行部署）。

四、博客部署过程

1．在MyBlog文件夹内右键点击Git Bash here并执行：

npm install hexo-deployer-git --save

2．打开Github中的已新建的项目,打开Clone or download复制项目地址:格式如https://github.com/Github账号名/Github账号名.github.io.git

3．打开MyBlog文件夹中的-config.yml(可以下载一个Notepad++或者其他编译器打开),修改最下方的deploy为:

deploy:
  type: git
  repo: https://github.com/Github账号名/Github账号名.github.io.git
  branch: master

注意有一个空格，修改后保存！！！

4．再运行如下命令行:

hexo g -d

至此,你就可以通过网址:Github账号名+github.io访问属于你自己的博客啦!!!

五、如何写博客文章

1．在Markdown官网下载Markdownpad2并安装

Markdown其实就相当于一个Office Word的一个编辑软件，学习和使用很容易掌握，具体语法可参考Markdown教程。

2．通过Markdownpad书写你的博客并放在MyBlog/source目录下

3．通过hexo g和hexo s在本地https://localhost:4000预览，再通过`hexo d`部署到你的博客网站上去，博客书写完成！！！
