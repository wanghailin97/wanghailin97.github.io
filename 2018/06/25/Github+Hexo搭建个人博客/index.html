Github+Hexo搭建个人博客

Posted on 2018-06-25 |

本文详细地说明了如何运用开源框架Hexo和使用Github免费的云空间完成个人免费博客的搭建，除此之外，还包括许多Hexo主题进行博客个性化设计的步骤．

一、知识了解

####　1．什么是Hexo

Hexo中文网站是一个快速，简洁且高效的博客框架．让上百个页面在几秒内瞬间完成渲染．Hexo支持Github　Flavored　Markdown的所有功能．甚至可以整合Octopress的大多数插件．并且自己也拥有强大的插件系统．简单理解Hexo就是一个开源的博客设计框架，个人可以随意进行下载、修改和设计．

####　２．什么是GIt和Github

通俗地解释，git是一种版本控制系统．跟svn、cvs是同级的概念．
github是一个网站，给用户提供git服务．这样你就不用自己部署git系统，直接用注册个账号，用他们提供的git服务就可以．可以理解为Git是在本地电脑上处理各种文件，而Github就像百度云盘云盘一样．更详细的解释可以参见大白话解释 Git 和 GitHub．

####　3．其他

诸如Node.js等都可以理解为一种程序语言解释器，不需要刻意掌握．

二、配置准备

####　1．Git

在Git官网下载属于本电脑配置的Git安装包并安装Git，安装过程比较简单，如有问题请参见Git安装配置教程．

####　2．Github

在Github官网注册Github账号，然后新建一个项目（PS：项目名称必须为:Github账号名+github.io，这也是设计的博客网址）．

####　3．Node.js

在Node.js官网下载后缀为LTS的安装包并安装．
